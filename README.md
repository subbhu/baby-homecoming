## How to set up

1. **Create the repository**
   - Create a new GitHub repository (for example, `baby-homecoming`).
   - Add these files to the root of the repository.
   - Rename `baby_homecoming_norsvp.html` to `index.html` so GitHub Pages serves it by default.

2. **Enable GitHub Pages**
   - Go to **Settings â†’ Pages** in your repository.
   - Under "Source", choose the `main` (or `master`) branch and the `/ (root)` folder.
   - Save. After a few minutes, GitHub will give you a public URL like:
     `https://<your-username>.github.io/baby-homecoming/`.

3. **Test the page**
   - Open the GitHub Pages URL.
   - Verify that:
     - All images load correctly.
     - The "When & where" section shows the correct address and Google Maps link.
     - The **Add to calendar** button triggers download of `baby-homecoming.ics` and opens it in your calendar app.

4. **Sharing the link**
   - Share the GitHub Pages URL with friends and family.
   - Anyone with the link can view the page and download the calendar invite.

## Editing text later
